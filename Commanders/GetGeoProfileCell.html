Cell Name GetGeoProfileCell
Cell Description GetIPCell job is to return Commander GeoProfile like: Country, State, City based on his IP Address
Input: String: IP Address
Output: GeoProfile Complextype: 'address' => array ( 'name' => 'address', 'type' => 'xsd:string' ), 'country' => array ( 'name' => 'country', 'type' => 'xsd:string' ), 'region' => array ( 'name' => 'region', 'type' => 'xsd:string' ), 'city' => array ( 'name' => 'city', 'type' => 'xsd:string' )
